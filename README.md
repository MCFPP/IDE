# Building from source

## Step 1. Clone the repository

Do this any way github lets you

## Step 2. Install dependencies

⚠️ Make sure to be inside the repository directory!

```
yarn install
```

## Step 3. Run/Build the app

```
yarn electron:build
```
or
```
yarn electron:serve
```
